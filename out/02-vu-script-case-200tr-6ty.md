## TIPS SẢN XUẤT

| Hạng mục | Yêu cầu |
|---|---|
| **Format** | Quay dọc 9:16 — TikTok/Reels/Shorts |
| **Ánh sáng** | Tự nhiên kết hợp đèn ấm (3200–3500K). Tránh ánh trắng lạnh — không phù hợp banker. |
| **Camera** | iPhone 14 trở lên hoặc Sony Alpha. 4K 30fps. |
| **Nhịp dựng** | Cắt nhanh — talking head + b-roll + biểu đồ + text pop-up. Mỗi cảnh tối đa 5s. |
| **Text overlay** | Lớn, màu tương phản (đỏ – vàng – xanh lá trên nền carbon). Font sans-serif (Be Vietnam Pro / Inter). |
| **Âm thanh** | Đoạn vấn đề: nền căng thẳng (heartbeat / string staccato). Đoạn kết quả: chuyển sang tích cực (piano + strings ấm). |
| **Nhạc nền** | Truyền cảm hứng — build up theo cảm xúc. Volume −18 dB so với voiceover. |
| **Microphone** | Lavalier (Rode Wireless GO 2) — tránh mic shotgun trong phòng có echo. |
| **Background** | Tối, có chiều sâu (bokeh đèn vàng) — không quá sáng, không có chữ phân tâm. |
| **Trang phục** | Polo trắng + quần tối, hoặc sơ mi xanh navy. Tránh kẻ sọc nhỏ (gây nhiễu camera). |
| **Đạo cụ** | Có thể cầm bút / tablet ở vài cảnh giải pháp — tăng cảm giác chuyên gia. |
